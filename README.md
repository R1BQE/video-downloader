# Video \& Audio Downloader

Универсальный скрипт на Python для скачивания видеоконтента и автоматической конвертации его в аудиоформат MP3.



## Основные возможности

* Два режима работы: скачивание видео в лучшем качестве или автоматическое извлечение аудиодорожки.

* Умная конвертация: использование ffmpeg для создания высококачественных MP3-файлов.

* Звуковые уведомления: программа проигрывает success.wav по завершении задачи, что удобно для мониторинга процесса.

* Доступность: интерфейс консольный, что прекрасно читается экранными дикторами.



## Требования

1\. Python 3.x.

2\. yt-dlp: библиотека для загрузки контента.

3\. FFmpeg: необходим для работы функций конвертации. Должен быть добавлен в системный PATH.

4\. Файл звука: положите любой звуковой файл с названием success.wav в папку со скриптом.



## Установка и запуск

1\. Склонируйте репозиторий или скачайте файл main.py.

2\. Установите зависимости:

```bash:
pip install -r requirements.txt
```



3\. Запустите скрипт:

```bash:
python main.py
```



## Как использовать

После запуска выберите нужный пункт меню (1 или 2) и введите ссылку на видео. Программа сама выполнит загрузку, конвертацию и подаст звуковой сигнал по окончании.

Поддерживает скачивание с youtube, rutube, vk видео, ну и вообще отовсюду, откуда может скачать yt-dlp.
